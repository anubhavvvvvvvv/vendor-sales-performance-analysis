# 📊 Enterprise Vendor Analytics using SQL & Python

## Overview

This project performs end-to-end analytics on a large-scale enterprise procurement dataset using SQL and Python. The workflow simulates a real-world analytics pipeline, starting from raw data ingestion through advanced SQL analysis, statistical validation, and interactive visualizations.

---

## Project Scale

- 📁 **7+ relational tables**
- 📦 **Millions of records processed**
- 🗃️ Large-scale procurement and inventory dataset
- 🔄 End-to-end ETL/EDA workflow
- 📈 Multiple business dashboards and statistical analyses
